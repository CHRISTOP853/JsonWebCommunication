# JSON Query Tool (WPF)
A desktop JSON tool built with WPF and .NET

## Features

Load JSON from:
    File
    URL
    Paste JSON
Parse and visualize JSON in a TreeView
Run path-based queries
Pretty Print JSON
Dynamically add  notes 

## Tech Stack

    .NET
     WPF
     System.Text.Json

## Run locally  in bash

    dotnet build
    dotnet run
